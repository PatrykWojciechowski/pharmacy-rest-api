# Pharmacy RESTful Web Service

This application is simple REST API	allowing users to perform CRUD operations on the database.

## About the app

To develop this application i used the following technologies:
- Spring (MVC, ORM)
- Hibernate ORM
- Jackson
- MySQL database
and more...
